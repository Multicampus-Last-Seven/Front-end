## 1. 구현한 핵심 기능

- 회원가입
- 로그인
- CCTV 등록/삭제
- MQTT를 통한 CCTV 영상 스트리밍

<br/>

## 2.  실행 방법

### a. 필요한 라이브러리 설치

```
yarn
```

<br/>

### b. 실행하기

```
yarn start
```
